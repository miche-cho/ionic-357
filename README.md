# ionic-357
BuzzCycle biking app


## Requirements
Node.js: https://nodejs.org/en/

Install Ionic: `npm install -g ionic`

## Steps to build on the web (recommended)

1.  `npm install`
2. `ionic serve`

To build on android: 
`ionic cordova build android`   ( Need android studio sdk )
